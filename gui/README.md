## What is it?

**ACTS UI** is the repository for Project ACTS' graphical user interface. It contains all the project configuration for Electron (underlying framework used for the application) and all the HTML, CSS, and JavaScript components of the application.

![ACTS Initial UI](external/images/gui.gif)

## Where to get it?

The source code is currently hosted in GitHub: [https://github.com/project-acts/acts-ui](https://github.com/project-acts/acts-ui)

## Dependencies

- [Node.js - a JavaScript runtime built on Chrome's V8 JavaScript Engine.](https://nodejs.org/en/)
- [Electron - a framework to build cross-platform desktop apps with JavaScript, HTML, and CSS](https://www.electronjs.org/)
- [Leaflet - an open-source JavaScript library for mobile-friendly interactive maps.](https://leafletjs.com/)
- [Material Design - Google’s most expressive and adaptable design system yet](https://material.io/)

## Installation from sources

To install and run `acts.ui` from source, locate and go in the **acts-ui** directory (same one where you found this file after cloning the git repository), execute:

```shell
$ npm install
$ npm start
```
